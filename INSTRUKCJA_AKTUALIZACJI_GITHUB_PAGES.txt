AKTUALIZACJA GITHUB PAGES DO WERSJI 2.1

Repozytorium: https://github.com/ethicsandschool-lgtm/etyczny-kompilator-ai
Strona: https://ethicsandschool-lgtm.github.io/etyczny-kompilator-ai/

1. Otwórz repozytorium i zakładkę Code.
2. Wybierz Add file -> Upload files.
3. Przeciągnij wszystkie pliki z tego folderu.
4. Zatwierdź zmianę bez tworzenia nowej gałęzi.
5. W polu opisu wpisz: Aktualizacja aplikacji do wersji 2.1.
6. Odczekaj 1–3 minuty i otwórz stronę.
7. Wykonaj twarde odświeżenie Ctrl+F5 albo Cmd+Shift+R.
8. Sprawdź: nową nazwę R2, szczegółowe porównanie zmian, opcjonalny prompt i stopkę „wersja 2.1”.
9. Wykonaj pełny test V1 -> V2 -> raport bez promptu oraz z promptem.

Adres strony i kod QR nie zmienią się. Nie zmieniaj nazwy repozytorium, gałęzi main ani ustawień Pages.
