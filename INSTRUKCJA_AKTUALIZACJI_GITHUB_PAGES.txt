AKTUALIZACJA GITHUB PAGES DO WERSJI 2.0

Repozytorium: https://github.com/ethicsandschool-lgtm/etyczny-kompilator-ai
Strona: https://ethicsandschool-lgtm.github.io/etyczny-kompilator-ai/

1. Otwórz repozytorium i zakładkę Code.
2. Wybierz Add file -> Upload files.
3. Przeciągnij wszystkie pliki z tego folderu.
4. GitHub poinformuje, że index.html, README.md i instrukcje zastąpią starsze wersje.
5. W polu opisu wpisz: Aktualizacja Etycznego Kompilatora AI do wersji 2.0.
6. Kliknij Commit changes.
7. Odczekaj 1–3 minuty i otwórz stronę.
8. Sprawdź, czy na ekranie widoczne są: S0, nowa R2 dotycząca osób ciężko poszkodowanych i stopka „wersja 2.0”.
9. Wykonaj pełny test V1 -> V2 -> raport.

Adres i kod QR nie zmienią się. Nie zmieniaj nazwy repozytorium, gałęzi main ani ustawień Pages.
